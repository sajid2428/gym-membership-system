# Gym Management System (Java)

This is a console-based Gym Management System developed using Java and Object-Oriented Programming principles.

The application allows users to create profiles, calculate calorie requirements, track strength records, and manage fitness-related information through a menu-driven interface.

---

## Features

- User registration and login system
- BMI (Body Mass Index) calculation
- BMR (Basal Metabolic Rate) calculation
- Maintenance calorie estimation
- Goal-based calorie recommendation (Bulk / Lean Bulk / Cut)
- Daily protein estimation
- PR tracking (Bench Press, Squat, Deadlift)
- 1 Rep Max (1RM) calculator
- Profile updates (weight and activity level)

---

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (ArrayList)
- Scanner for user input handling

---

## Formulas Used

### BMR (Mifflin-St Jeor Formula)

For Male:
BMR = (10 × weight) + (6.25 × height) − (5 × age) + 5

For Female:
BMR = (10 × weight) + (6.25 × height) − (5 × age) − 161

### 1 Rep Max (Epley Formula)

1RM = weight × (1 + reps / 30)
