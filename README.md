# Gym Membership Management System

- A Java-based console application designed to manage gym memberships efficiently.  
This project demonstrates core Java concepts like OOP, classes, objects, and user input handling.

##  Features

 - Add new gym member
 - View member details
 - Update membership information
 - Remove member
 - Calculate membership details (if applicable)

## Technologies Used

 - Java
 - Object-Oriented Programming (OOP)
 - Classes & Objects
 - Scanner (User Input Handling)
